Position static relative absolute fixed sticky

  1. Position: Static

    2. ~ Default position. Cannot change its place.

  2. Position: Relative

    3. Relative to its own position. Can change its place but also don't disappear from its own position.

  3. Position: Absolute

    4. Absolute to the parent element. It completely disappears from it original place to change position.

  4. Position: Fixed
  5. Position: Sticky
